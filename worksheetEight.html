Ethical Implications of Deceptive (dark) patterns

Hidden Subscriptions

Software is not always free to use or interact with and can require a user to pay a subscription to use the product. However, software that does not alert or bring awareness to the product being a pay to use can be deceptive to the user, the software even will allow the user to use the software and then send the user charges for their usage. Recurring subscriptions that do not explicitly say it's recurring or alert the user of an upcoming charge are deceptive dark patterns. Ethical Implications of Deceptive (dark) patterns violate the ethical principles set by AMC.

AMC Code of Ethics and Professional Conduct

1.2 Avoid harm. - This code is being broken as well as unwanted fees can be harmful to those living on a tight budget as hidden subscriptions are practically stealing. This leads to unjust mental harm by bringing unwanted stress. As money is the owners property, hidden subscriptions are causing unjust damage to private property.
